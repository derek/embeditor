Embeditor

A super easy way to embed a text editor onto any webpage.

Ok. How?

Simple.

            
    <iframe src="http://embeditor.org/"></iframe>

That's it! To make it bigger/smaller, use CSS to resize. The editor will take up 100% height/width of the iframe.

For example


    <iframe id="embeditor" src="http://embeditor.org/"></iframe>
    <style>
        #embeditor{
            height:500px;
            width:300px;
            overflow: hidden;
            border:none;
        }
    </style>

API

  • Custom modes - Embeditor currently support HTML, CSS, and JS. More on the way. To activate, simply append ?mode=javascript, ?mode=html, or ?mode=css to the URL.

    
    <iframe src="http://embeditor.org/?mode=javascript"></iframe>
